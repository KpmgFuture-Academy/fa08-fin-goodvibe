# 타이포그래피 시스템

## 폰트 패밀리

```css
font-family: "Pretendard", "Noto Sans KR", -apple-system, BlinkMacSystemFont, sans-serif;
```

- **Pretendard** (1순위) — 한국어 가독성 우수, 모던
- **Noto Sans KR** (2순위) — Google Fonts, 광범위한 호환
- 시스템 폴백 — Apple/Windows 기본
- **CDN 자동 로드** 가정. CSP 또는 오프라인 환경에선 self-host 필요

## 기본 크기

```css
--font-base: 18px;       /* :root 변수, body 기본 */
line-height: 1.5;        /* body 기본 */
```

> **왜 18px?** — 고령 이장님 사용자 대상. 일반 web 16px 대비 한 단계 큼. UI 텍스트도 이 기준으로 ±2~6 변동.

## 크기 스케일

`globals.css` 에서 실제 사용 중인 크기. **자주 쓰는 굵기 + 용도** 묶음.

| 크기 | 굵기 | 용도 (대표 위치) | 예 |
|---:|---|---|---|
| **36px** | 800 | 큰 KPI 숫자 (StatCard value) | `34건` |
| **28px** | 800 | 페이지 타이틀 (PageHeader h1) | "참여사업관리" |
| **23px** | 700 | 카드 내 큰 통계값 | `15만원` |
| **22px** | 800 | 모달 헤더 (Modal title), 상세 강조 | "현재 할 일" 영역 |
| **20px** | 800 / 700 | 사이드바 브랜드 / 카드 헤더 제목 (CardHead) | "마을 주민" |
| **19px** | 700 | 큰 버튼 (`btn-lg`) | `[저장하기]` |
| **18px** | 400 / 600 / 700 | 본문 기본 (body) / 강조 텍스트 | 일반 문장 |
| **17px** | 700 / 500 | 페이지 부제, 라벨 강조 | 부제 |
| **16px** | 600 / 700 | 카드 섹션 제목, 사이드바 메뉴 라벨, 버튼(`btn-md`) | "농가별 이행 현황" |
| **15px** | 700 / 600 | 본문 중간 (모달 본문 등) | 모달 안 텍스트 |
| **14px** | 600 / 400 | 표 본문, 보조 액션 (작은 버튼 `btn-sm`) | 테이블 cell |
| **13px** | 400 | 라벨, 부가 설명 (`.muted` 자주) | "참여 농가 수" |
| **12px** | 400 / 500 | 사이드바 라벨, 진척률 보조 (`N/M`) | "현재 마을" |

### 굵기 (font-weight) 의미

| Weight | 의미 | 사용 |
|---:|---|---|
| **800** | 헤드 (제목, KPI 값) | 페이지 타이틀, 큰 숫자 |
| **700** | 강조 (subheader, primary 버튼) | 카드 제목, 활성 메뉴 |
| **600** | 중간 (라벨, secondary 버튼) | 작은 제목, 라벨 |
| **500** | 약한 강조 (input 텍스트) | 메뉴 항목 |
| **400** | 본문 기본 | 일반 단락 |

## 자간 (letter-spacing)

대부분 기본(0). **예외 4 곳**:

| 영역 | 값 | 이유 |
|---|---|---|
| 사이드바 브랜드 ("저탄마을") | `0.04em` | 로고 느낌 — 약간 트래킹 |
| 사이드바 마을 라벨 ("현재 마을") | `0.08em` | 작은 라벨 가독성 (12px 이하) |
| primary 버튼 텍스트 | `0.01em` | 가독성 약간 |
| 그 외 | `normal` (`0`) | — |

---

## 행간 (line-height)

| 영역 | 값 | 이유 |
|---|---|---|
| body 본문 | `1.5` | 다단락 가독성 |
| 페이지 타이틀 (큰 글씨) | `1.2` | 줄간 압축 |
| 큰 KPI 숫자 (36px) | `1.05` | 시각 무게중심 |
| 사이드바 부제 / 카드 보조 | `1.45` | 약간 압축 |

## 시맨틱 HTML 매핑

| HTML 태그 | 적용 클래스 또는 스타일 | 결과 |
|---|---|---|
| `<h1>` | `.page-title` | 28px / 800 / 1.2 |
| `<h2>` | `.section-title`, `.modal-title` | 22px / 700 |
| `<p>` | (없음 — body 상속) | 18px / 400 |
| `<th>` | (table 안) | 14px / 600 |
| `<td>` | (table 안) | 14px / 400 |

## 사용 예시

```tsx
{/* 페이지 헤더 */}
<PageHeader title="참여사업관리" sub="마을이 참여 중인 사업 3개" />
// → h1 28px/800 + p 17px/400

{/* 카드 안 큰 강조 */}
<div style={{ fontSize: 22, fontWeight: 800 }}>중간 물떼기</div>

{/* 라벨 */}
<div className="muted" style={{ fontSize: 13 }}>현재 할 일</div>

{/* KPI */}
<StatCard label="참여 농가" value="8명" />
// → label 13px muted, value 36px/800
```

## 안티 패턴

- ❌ **임의 크기**: `fontSize: 21px` — 위 스케일에 없음. `22px` 또는 `20px` 로 통일
- ❌ **inline `color`**: `style={{ color: "#333" }}` — `var(--ink-soft)` 또는 `.muted` 사용
- ❌ **font-family 재정의**: `font-family: "Helvetica"` — 시스템 전체에서 Pretendard 유지

## 새 크기 추가가 필요할 때

신규 스케일이 정말 필요하다면:
1. 디자이너/팀과 합의
2. globals.css 의 컴포넌트별 스타일에 추가
3. 본 문서 표 갱신 + 용도 설명
4. **임의값** 사용 금지 — 다음 합류자가 추적 가능해야 함
