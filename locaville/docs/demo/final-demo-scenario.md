## Part 1. 농업인 앱 — 김영수 (2분)

### 1) 진입
- 폰 Chrome → 농업인 앱 URL
- Splash (저탄마을) → 로그인 화면 → **카카오톡으로 로그인**
- 홈 로딩 (새싹 트랜지션 2.2초) → 홈 화면

### 2) 홈 화면 — 한 화면에 모든 게
- 헤더: 6월 X일 화요일 + 날씨 (구름 많음 22°↑29 ↓20)
- **✦ AI 오늘 한마디** — "김영수님, 구름 많고 23도예요. 농자재 구입 후 사진 한 장 꼭 남겨두세요."
- **오늘 할 일** 카드:
  - "농자재 구입" 일러스트 + "⚠ 6월 30일까지 해야해요!"
  - 큰 초록 버튼 **[📷 영수증 찍고 완료하기]**
  - 우측 "N건 더 보기 ›"
- **오늘 있었던 일도 남겨두세요** — [🎙 말로 남기기] / [✎ 손으로 직접 적기]
- **농사 도와주기** 카드 — "박정호님 농가로 이동" (helper 모드)

### 3) 사진 등록 흐름
- **[영수증 찍고 완료하기]** 클릭
- PhotoLiveCoachOverlay 풀스크린
  - HTTPS 환경: 카메라 자동 시작 → 3초마다 "평평하게 펴 주세요" / "잘 보여요, 찍어 주세요" TTS
  - LAN HTTP: file input fallback (갤러리 또는 카메라 다이얼로그)
- 셔터 → **CompletionModal**
  - 큰 ✓ + "사진을 등록했어요"
  - 부제 "이장님이 한 번 더 확인하실 거예요"
  - **AI 가 본 사진** chip + 분류 결과
  - 위치 (역지오코딩 주소)
  - **[홈으로 가기]**

### 4) 자유 기록 (선택)
- "오늘 있었던 일도 남겨두세요" → [✎ 손으로 직접 적기]
- ManualInputScreen
  - 카드 1: 날짜 + 필지 + 작물
  - 카드 2: 작업 종류 chip
  - 카드 3: 세부 작업내용 textarea
  - **카드 4: 사진 첨부 (선택)** — [📷 사진 추가하기]
- 사진 첨부 시 PhotoLiveCoachOverlay 다시 → 셔터 → 미리보기
- **[기록 저장하기]** → CompletionModal

## Part 2. 이장님 대시보드 (2분)

### 1) 진입
- 폰 또는 PC 브라우저 → `web_user/dashboard`
- 헤더: "저탄마을 대시보드 / 오늘 먼저 챙길 농가와 증빙을 확인하세요"

### 2) 오늘 먼저 챙길 일 (메인)
- 큰 카드 + 6px accent 좌측 띠
- numbered list ①②③
  - ① **박정호 농가** — "논물 얕게 걸러대기 사진이 아직 없어요" + 마감일 + [문자로 알려주기] [농가 보기]
  - ② 김영수 농가 — "농자재 구입 사진 증빙이 필요해요" + ...
  - ③ 이민호 농가 — "최근 7일 기록이 적어요" + [농사 도와주기 연결]

### 3) 즉시 행동 — 문자 알림
- ① 박정호 옆 **[문자로 알려주기]** 클릭
- 즉시 버튼이 **"✓ 알림을 보냈어요"** chip 으로 전환 (accent border + 배경)
- 0.3초 후 자동 재정렬 — 박정호가 ③ 또는 사라짐, ④번이 ① 자리로

### 4) 보조 정보
- KPI 4종: 오늘 사진 / 전체 남은 일정 / 이번 주 마감 / 도움 필요한 농가
- **챙겨야 할 농가** 리스트 (top 3 외 나머지)
- **오늘의 운영 메모** (한 줄 + 들어보기)
- **농사 도와주기 연결** — 박정호 ← 김민수 도우미 / 양쪽 승인 대기
- **최근 올라온 사진** 작은 썸네일 6장
- **마을 일정** 캘린더

## Part 3. 농사 도와주기 (helper mode) (1.5분)

### 1) 김영수 폰으로 다시 (도우미 역할)
- 홈 화면 하단 **"농사 도와주기"** 카드:
  - "박정호님 농가로 이동" + "양쪽 승인 완료"
  - 클릭

### 2) 트랜지션
- **HelperModeTransitionScreen ("enter")** 풀스크린
- 트랙터가 좌→우로 이동 (2.5초)
- 그 사이 backend 가 박정호의 todo + 사진 + 일지 fetch

### 3) 박정호 농가로 진입
- 헤더 띠 — 짙은 초록색 "박정호님 기록 도와드리는 중"
- 홈 화면이 박정호의 데이터로 표시
- todo 카드 → 박정호의 활동
- 사진/일지 기록 시 backend 가 박정호의 `user_no` 로 저장

### 4) 돌아오기
- 헤더 띠의 **"도움 마치기"** 버튼
- HelperModeTransitionScreen ("leave") — 트랙터 우→좌
- 동시에 김영수 데이터 fetch
- 트랜지션 끝 → fully loaded 김영수 홈
