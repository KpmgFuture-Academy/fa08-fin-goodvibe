# 현재 실제 구현 기능 요약

- 분석 대상: `locaville/backend`, `locaville/app_user`, `locaville/web_user`
- 작성 기준: 코드상 화면, API client, Backend router/service 근거가 확인되는 기능만 포함
- 제외 기준: 코드 근거가 없는 가입 플로우, 외부 본인인증, OCR 보정, 주민 삭제, 제출 ZIP 생성 등은 제외
- 민감정보: `.env` 및 키/비밀번호/접속 문자열은 읽거나 포함하지 않음
