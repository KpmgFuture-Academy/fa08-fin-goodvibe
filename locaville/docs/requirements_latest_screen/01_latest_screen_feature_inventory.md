# 최신 화면 기준 기능 인벤토리

- 분석 기준: `locaville/app_user`, `locaville/web_user`의 최신 화면/메뉴/버튼/사용자 흐름
- Backend 확인 용도: 화면에서 호출하는 API endpoint, router, service/repository 근거 확인
- 제외 기준: 기존 요구사항 정의서 내용은 참고하지 않음
- 보안 기준: `.env`는 열거나 출력하지 않음, 민감정보는 포함하지 않음
